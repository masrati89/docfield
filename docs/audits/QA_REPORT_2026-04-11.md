## 🔴 חוסמים (Critical Blockers)

### BUG-01 — הפקת PDF נשברת כליל ב-Web

- **מסך:** `reports/[id]` → "הורדת דוח" או "תצוגה מקדימה" → "הפק PDF"
- **שחזור:** לחיצה על "הפק PDF" בתוך bottom-sheet "סיכום דוח"
- **התנהגות מצופה:** פתיחת PDF / דיאלוג הדפסה / הורדת קובץ
- **התנהגות בפועל (ריצה א׳):** הזרקה של שגיאה קטלנית ב-console:
  ```
  [WARNING] @supabase/gotrue-js: Lock "lock:sb-127-auth-token" was not released within 5000ms
  [WARNING] @supabase/gotrue-js: Lock "lock:sb-127-auth-token" was not released within 5000ms
  [ERROR] Uncaught Error: Lock broken by another request with the 'steal' option.
  ```
  ה-hook `usePdfGeneration` מבצע מספר קריאות אוטנטיקציה במקביל (`supabase.auth.getSession()` + RLS query) שנועלות את ה-WebLock של gotrue-js פעמיים → lock נשבר → ה-promise נזרק → כל תהליך ההפקה נופל.
- **התנהגות בפועל (ריצה ב׳, אחרי רענון):** אין שגיאה, **אבל גם אין PDF** — ה-sheet נסגר בשקט, אין window.print, אין הורדה, אין toast הצלחה. expo-print על web לא נקשר לפעולה כלשהי.
- **השלכה:** הפיצ׳ר המרכזי של המוצר לא עובד ב-Web כלל.
- **תיקון מוצע:**
  1. לרכז את כל קריאות ה-auth ב-`usePdfGeneration` למשיכה אחת של session ולהעביר אותה לכל הקריאות הבאות (ללא re-fetch).
  2. ב-Web — להפריד פלטפורמה: `Platform.OS === 'web'` → שימוש ב-`printAsync` + fallback ל-`window.open(html, '_blank')` + `window.print()`, במקום ה-expo-print המקורי שלא שולח event במצב dev.
  3. להוסיף `try/catch` סביב `Print.printToFileAsync` עם toast בעברית במקום שגיאה שקטה.

### BUG-02 — Pre-PDF Summary נפתח מאחורי ה-Preview Modal

- **מסך:** `reports/[id]` → "תצוגה מקדימה" → "הפק PDF"
- **בעיה:** ה-bottom-sheet "סיכום דוח" נפתח עם `zIndex: 0` ולכן מוסתר על ידי modal ה-preview שנמצא ב-`zIndex: 999`. המשתמש לא רואה דבר ונדמה לו שהכפתור לא עובד.
- **תיקון:** ב-`components/checklist/ReportPreviewSheet.tsx` להעלות את ה-zIndex של PreSummarySheet ל-1000+, או לסגור את ה-preview אוטומטית לפני פתיחת ה-summary.

### BUG-03 — Verify Email חוסם משתמשי בדיקה

- **מסך:** `/(auth)/login` → אחרי login מוצלח
- **בעיה:** משתמש שנוצר ידנית דרך API מופנה ל-`/(auth)/verify-email` ונחסם לצמיתות. `email_confirmed_at` הוא generated column ולא ניתן לעדכון ישיר — רק `confirmed_at`. אין seed ל-QA.
- **תיקון:** להוסיף seed של משתמש QA מאומת ב-`supabase/seed.sql`, או endpoint `/dev/login-as` בסביבה מקומית בלבד.

## 🟠 תקלות פונקציונליות

### BUG-04 — "שיתוף" ו"הגדרות דוח" מחוברים לאותו bottom-sheet

- **מסך:** `reports/[id]` → SubHeader
- **בעיה:** כל 4 הכפתורים (הורדת דוח / הגדרות דוח / שיתוף / תצוגה מקדימה) פותחים אותה bottom sheet של "סיכום דוח". "שיתוף" אינו מפעיל `Share API` של המערכת, "הגדרות דוח" לא פותח הגדרות.
- **תיקון:** כל כפתור מחייב handler נפרד ב-`ReportActionsBar.tsx`:
  - שיתוף → `navigator.share({ files: [pdfBlob] })` ב-web, `Sharing.shareAsync` במובייל.
  - הגדרות דוח → ניווט ל-`reports/[id]/settings` או פתיחת `ReportSettingsSheet`.

### BUG-05 — "קומה: undefined" ב-Preview

- **מסך:** `reports/[id]` → תצוגה מקדימה
- **בעיה:** שדה קומה מוצג כ"קומה: undefined" במקום `apartments.floor` או "לא צוין". ה-join עשוי להחזיר `null` ואין `??` בקוד.
- **תיקון:** `ReportPreviewSheet.tsx` → `apt?.floor ?? 'לא צוין'`.

### BUG-06 — תאריך ISO גולמי ב-Preview

- **מסך:** `reports/[id]` → תצוגה מקדימה
- **בעיה:** מוצג `2026-04-11T22:14:38.123Z` במקום `11.04.2026`. אין formatter.
- **תיקון:** להשתמש ב-`formatDateHe()` מ-`packages/shared/src/utils/formatters.ts` (כבר קיים ובשימוש במקומות אחרים).

### BUG-07 — חיפוי קטגוריה חסר ב-DefectRow במסך Add Defect

- **מסך:** `reports/[id]/add-defect` → CategoryPicker
- **בעיה:** בחירת קטגוריה לא מתמשכת אחרי שמירה + חזרה — הערך נמחק מה-state בגלל `resetForm()` שנקרא אחרי `onBlur` של השדה ולא אחרי submit בלבד.
- **תיקון:** להעביר את `resetForm` ל-`onSuccess` של המוטציה בלבד.

### BUG-08 — ChangePasswordSection: PasswordField Ref Warning

- **מסך:** `settings` → שינוי סיסמה
- **בעיה:** `Warning: Function components cannot be given refs. Attempts to access this ref will fail.` — `PasswordField` הוא functional component ללא `forwardRef` אבל `ChangePasswordSection` מעביר אליו ref ל-focus.
- **תיקון:** לעטוף ב-`React.forwardRef` + `useImperativeHandle({ focus })`.

### BUG-09 — Defect Library לא מתמשך בחזרה לטופס

- **מסך:** `add-defect` → פתיחת ספריית ליקויים → בחירה
- **בעיה:** פריט נבחר, modal נסגר, אך ערכי השדות בטופס לא מתעדכנים. ה-handler `onSelectLibraryItem` קיים אבל לא מחובר ל-`setValue` של ה-form.
- **תיקון:** ב-`useDefectLibrary.ts` להחזיר callback שמזין את כל 7 השדות.

### BUG-10 — Tab "חוסרים" מראה 8 אבל דירה חדשה

- **מסך:** `reports/[id]` → tab חוסרים
- **בעיה:** דירה חדשה שנוצרה לטסט מציגה "8" חוסרים לפני שבוצע checklist. הספירה מחושבת מה-template ולא מ-`checklist_results` בפועל → counter שגוי לפני שיש נתונים.
- **תיקון:** `useReport.ts` → לספור `checklist_results.filter(status === 'missing')` במקום את כל הפריטים בתבנית.

## 🟡 UX / Visual

### BUG-11 — FAB של Reports List על חלק התחתון חותך ב-tab bar

- הכפתור `bottom: 80` אבל ה-tab bar על web גובה 64+safe-area → חופף בכ-8px.

### BUG-12 — Side Menu נפתח משמאל במקום מימין

- בניגוד לכלל מ-memory: ההמבורגר/side-menu חייבים להיות RIGHT. בדיקה ראתה slide משמאל ב-`SideMenu.tsx`.
- **תיקון:** `translateX` מ-`-300` ל-`+300`, Pressable ב-`start: 0` ולא `end: 0`.

### BUG-13 — Tab bar 4-tabs נראה גם ב-delivery reports

- בניגוד ל-memory: 4-tab bar הוא רק ל-bedek_bait. בדיקה מצאה אותו גם ב-`protocol_mesira`.
- **תיקון:** `reports/[id]/_layout.tsx` → `if (report.type === 'protocol_mesira') hideTabBar`.

### BUG-14 — CategoryAccordion: סמל חץ לא מסתובב בפתיחה

- `useAnimatedStyle` קיים אבל `rotate` תלוי ב-`isOpen` שלא מועבר כ-shared value.
- **תיקון:** להעביר `withTiming(isOpen ? 180 : 0)` ולא boolean מקומי.

### BUG-15 — Empty State של Projects ללא CTA

- "אין פרויקטים עדיין" מוצג בלי כפתור "צור פרויקט חדש".
- **תיקון:** להוסיף `Button` עם ניווט ל-wizard.

### BUG-16 — Reports List: pull-to-refresh לא מופעל ב-Web

- `RefreshControl` לא נתמך מלא ב-RN-Web, אין fallback.
- **תיקון:** להוסיף כפתור "רענן" ב-header במצב `Platform.OS === 'web'`.

## ⚪ אזהרות Console (לא חוסמות אבל מצביעות על חוב טכני)

### WARN-01 — `"shadow*" style props are deprecated. Use "boxShadow"`

- מופיע בכל מסך. יש `shadowColor/shadowOffset` ב-~15 קומפוננטות.
- **תיקון:** החלפה ל-`boxShadow: '0 4px 16px rgba(60,54,42,.1)'` (תואם גם למה שכבר קיים ב-FAB).

### WARN-02 — `props.pointerEvents is deprecated. Use style.pointerEvents`

- שימוש ישן ב-`pointerEvents="none"` על `View`.

### WARN-03 — `Cannot manually set color scheme, as dark mode is type 'media'`

- `expo-router` מנסה לקבוע dark mode ידנית אבל `app.json` מגדיר `"userInterfaceStyle": "automatic"` דרך media query.
- **תיקון:** `StyleSheet.setFlag('darkMode', 'class')` ב-`_layout.tsx` הראשי.

- **Login/Register flow** (אחרי seed)
- **ניווט Tab bar** בין 4 המסכים הראשיים
