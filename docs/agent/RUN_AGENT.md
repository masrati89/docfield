## טיפים

**לשיחה יעילה:**

- תמיד ציין את context — "אני בונה את Home Screen" ולא "אני בונה משהו"
- אם יש mockup — אמור את שמו המדויק
- אחרי כל Task שמסתיים — עדכן את CONTEXT.md ידנית

**סדר עבודה מומלץ:**

```
1. הרץ Agent → בקש דוח מצב
2. Agent מציג Backlog
3. בחר Task → Agent מייצר TASK_XXX.md
4. הרץ את TASK_XXX.md ב-Claude Code נפרד
5. בדוק תוצאה → עדכן CONTEXT.md
6. חזור ל-1
```
