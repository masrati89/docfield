### Data Collection

| Data Type    | Collected | Purpose                  | Optional |
| ------------ | --------- | ------------------------ | -------- |
| Name         | Yes       | Account identification   | No       |
| Email        | Yes       | Account identification   | No       |
| Phone Number | Optional  | Professional profile     | Yes      |
| Photos       | Yes       | Defect documentation     | No       |
| Device ID    | Yes       | Crash reporting (Sentry) | No       |
| Location     | No        | N/A                      | N/A      |
| Calendar     | No        | N/A                      | N/A      |
| Contacts     | No        | N/A                      | N/A      |

### Data Security

- **Encryption in transit:** TLS 1.2+
- **Encryption at rest:** AES-256 (Supabase)
- **Data retention:** Active account indefinitely; 30 days after deletion; legal reports 7 years
- **Third-party sharing:** Only Supabase (processor), Apple OAuth, Google OAuth, Sentry (non-PII)
- **User rights:** Can request deletion, data export, right to forget at any time

## Permissions Explanation

### Camera (`android.permission.CAMERA`)

- **Why:** Take photos of defects for documentation

### Photos (`android.permission.READ_EXTERNAL_STORAGE`, `WRITE_EXTERNAL_STORAGE`)

- **Why:** Select & save report attachments

### Internet (`android.permission.INTERNET`)

- **Why:** Sync data with servers, authenticate user

### Network State (`android.permission.ACCESS_NETWORK_STATE`)

- **Why:** Detect offline mode, sync when reconnected

### Contacts (Optional)

- **Why:** Not used; can be removed
