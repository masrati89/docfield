Before creating a new screen:
1. Read docs/DESIGN_SYSTEM_DOCFIELD.md for colors, typography, spacing
2. Read .claude/DESIGN_CONTEXT.md for brand context
3. Read the bencium MOTION-SPEC.md for animation timing and easing
4. Run: python3 .claude/skills/ui-ux-pro-max/scripts/search.py "$ARGUMENTS" --stack react-native
5. Plan the screen layout, components needed, and data flow
6. Present the plan to me and wait for approval
7. Then implement with full DocField design system compliance:
   - Cream backgrounds, green primary, warm shadows
   - RTL layout, Rubik font
   - Press animations, haptic feedback
   - Skeleton loading, staggered list animations
   - Error states and empty states designed
